# Drawing Project 3

A simple Java Swing GUI application that draws a figure using Graphics2D.

## Description

This application uses a custom JPanel to draw a simple face figure with a rectangle for the head, circles for eyes and ears, and an arc for the mouth, all drawn in red.

## How to Compile

Navigate to the project directory and compile the Java file:

```bash
cd "Java Projects/project 3 drawing/src"
javac DrawingApp.java
```

## How to Run

After compiling, run the application:

```bash
java DrawingApp
```

## Usage

Simply run the application, and a window will appear displaying the drawn figure.
